# Student-Course-Management-System
A C++ console application for managing student records, academic performance, and course registrations.

## Project Deliverables
- **GitHub Link:** [github.com/emanelghamry]
- **Documentation:** Comprehensive PDF report including Analysis, Design, and Testing.
## Core Features
- **Student Management:** Create, Read, Update, and Delete (CRUD) operations for student data.
- **Course Tracking:** Enroll students in unique courses using `std::set`.
- **GPA Ranking:** Automatic sorting of students based on their GPA.
- **Persistence:** Data is saved to and loaded from `students.txt` automatically.

## How to Compile and Run
1. Ensure you have a C++ compiler installed (GCC/Clang).
2. Clone the repository and navigate to the project folder.
3. Compile the code:
   ```bash
   g++ main.cpp -o StudentSystem
